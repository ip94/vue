Installing Vue.js is as simple as it sounds. However, there are a few ways to create a project once your server is ready. The method below is how I installed Vue on my Ubuntu server.

Step 1: NPM

$npm install vue

Step 2: CLI

Great for building single-page application on the run. Vue.org recommends prior knowledge in Node.js or active learning in their guide if using CLI.

$ npm install --global vue-cli
# create a new project using the "webpack" template
$ vue init webpack bikewhere
# install dependencies and go!
$ cd bikewhere
$ npm install
$ npm run build
